# JLKB 键盘对应VID PID
-----

## VID & PID 说明：

可定制任意配列键盘，JLKB官网 [jlkb.jlkb.top](https://jlkb.jlkb.top)

- [jlkb](/keyboards/jlkb/)
- VID 区分键盘型号
- PID 区分摇杆、game、key、mouse，依次增加。
- game-游戏摇杆、key-映射按键、mouse-模拟指点杆。

|     名  称        | VID     | PID    |
|-------------------|---------|--------|
| JLbox-game        | 0xF001  | 0x3001 |
| JLbox-key         | 0xF001  | 0x3002 |
| JLbox-mouse       | 0xF001  | 0x3003 |
| JLf12-game        | 0xF002  | 0x3001 |
| JLf12-game-e      | 0xF002  | 0x3004 |
| JLf16e3t1-game    | 0xF003  | 0x3001 |
| JLk42-game        | 0xF004  | 0x3001 |
| JLlefpad-game     | 0xF005  | 0x3001 |
| JLmpad-game       | 0xF006  | 0x3001 |
| JLcut-game        | 0xF007  | 0x3001 |
| JLk26e6-game      | 0xF008  | 0x3001 |
| JLBOX-game        | 0xF009  | 0x3001 |
